Formulário HTML

Um formulário HTML é feito de um ou mais widgets,esses widgets podem ser campos de texto (linha única ou de várias linhas), caixas de seleção, botões, checkboxes ou radio buttons. A maior parte do tempo, estes elementos são emparelhados com uma legenda que descreve o seu objetivo.

Entradas de formulario

: Cria um campo de texto onde os usuários podem inserir texto.

: Permite que os usuários insiram um endereço de email.

: Exibe um campo de entrada onde os caracteres inseridos são mascarados, útil para senhas.

: Exibe um seletor de data.

: Permite que os usuários selecionem uma ou várias opções de um conjunto.

: Permite que os usuários selecionem uma única opção de um conjunto.

Elementos de formulário HTML

  • input – Define um campo de entrada de dados;
  • textarea – Define uma área de texto, podendo conter diversas linhas de texto;
  • button>– Define um botão;
  • select – Define uma lista selecionável, também conhecida como drop-down;
  • option – Define uma lista de opções dentro de um drop-down;
  • optgroup – Define um grupo de opções;
  • fieldset – Define um grupo de campos;
  • label – Define um rótulo ou legenda para um campo ou controle do formulário;
  • legend – Define um título para o conjunto de campos.

O input do tipo “file” define um campo de seleção de arquivos e um botão para upload de arquivos.




O input do tipo “image” define uma imagem como botão de envio do formulário HTML.

O input do tipo “number” define um campo para inserir um número.

  • Max – especifica o valor máximo permitido;
  • Min – especifica o valor mínimo permitido;
  • Tep – especifica os intervalos numéricos legais.
